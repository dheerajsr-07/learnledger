Can it parse DOCX and PDF?

Yes. The static dashboard simulates parsing locally, and the included backend has real Multer upload routes for PDF/DOCX parsing.

Is my data secure?

The static demo stores data in your browser only. The production backend includes JWT protection, bcrypt password hashing, rate limiting, and input sanitization.

Does Get Started open login?

Yes. Get Started takes users to login. Successful login opens the dashboard.
